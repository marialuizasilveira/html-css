Formatações

Negrito

Usamos o strong para colocar em negrito com semântica

Itálico

Usamo o em para colocar em itálico com semântica

Marcar o texto

Usamos o mark para fazer um texto marcado

Podemos também trocar a cor do marca texto com o style ou então de todos os marca texto no site com css no head =

Texto pequeno

Usamos o small para fazer um texto pequeno com semântica

Texto deletado

Usamos o del para fazer um texto com um traço no meio dele, ou seja ele vai ser escrito mas não é para ser considerado

Texto inserido

Usamos ins para fazer um texto sublinhado, também chamado de inserido

Texto sub e sup

Para fazer um texto elevado, usamos sup. Para fazermos um inferior, usamos sub

sup: 24th March / X3+10

sub: H2O

Códigos

Usamos code para fazer textos mono espaçados. Para todos terem os mesmo tamanho de espaço.

Pré definido

Usamos pre pra que o texto fique no mesmo formato que escrevemos no estudio

        
Poema
            Gosto de maça
        Com laranja
                            E sem caroço

Citações Simples

Usamos q para fazermos citações. A simples é quando só colocará aspas. Ex: Eu amo chocolate

Citação exemplificada

Nessa, usamos blockquote para fazermos citações em um paragráfo diferente. Usamos também o parâmetro cite para fazer com que os direitos autorais permaneçam

Uma citação de um artista, escritos, pintor, entre outros.

Abreviações

Se quisermos usar a Abreviações, mas queremos que o usuario do site saiba o significado, usamos abbr. Ex: HTML

Texto invertido

Usamos bdo para fazer um texto invertido. Usamos o dir para escolhermos a direção - rtl (right to left) e ltr (left to right)
